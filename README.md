# SideQuest

## About This Project

This game was developed for my Final Year Project.

## Download and Playtest

1. Download the latest game build from the **Releases** section.
2. Extract the downloaded ZIP file.
3. Open the extracted folder.
4. Run the `.exe` file to start the game.

> Please keep all extracted files in the same folder. Windows may display a security warning because the game is not digitally signed.
