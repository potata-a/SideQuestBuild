# SideQuest

## About This Project

This game was developed for my Final Year Project.

## Download and Playtest

1. Download the latest game build from the **Releases** section.
2. Extract the downloaded ZIP file.
3. Open the extracted folder.
4. Run the `.exe` file to start the game.

## How to connect with each other
### On the Host Device
1. Open command prompt and run 'ipconfig'
2. Search for "IPv4 Address" and take note (e.g: 192.168.xx.xx or 172.20.10.xx)
3. Open the game, press the Host button
4. Start the game and have fun with your friends!

### On the Client Device
1. Type in the Host's IP Address in the input field
2. Press the Client button
3. Wait for the Host to start and enjoy the game!
